# RESUMIND

This application is an AI-driven resume analyzer and ATS scoring platform designed to optimize resumes for modern tracking systems.

- In the application, you can upload your resume to parse its text content seamlessly. 
- The platform breaks down evaluation metrics to evaluate formatting, keyword matching, and structure. 
- The goal of the app is to give actionable feedback and score optimization tips to maximize hiring chances.

>*The application is written using html, css, ts, react, react router.*

---


## Start-up instructions

*You can view the finished version of the site by clicking on the link:* (https://resumind-pi.vercel.app)

#### If you want to run the site on your computer, follow these instructions:
1. Make sure you have an LTS version of Node.js installed on your computer.
   [Download and install](https://nodejs.org/) it if necessary.
2. Clone this repository.
3. Using the terminal or IDE, navigate to the project folder.
4. Install the project's base dependencies with the `npm install` command.
5. Start the application by executing the `npm run dev` command.
6. Done! Go to the following address in your browser: [http://localhost:5173](http://localhost:5173).
